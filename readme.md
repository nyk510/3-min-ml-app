# 3 min ML App

Minimally configured machine learning application (from streamlit)

## Requirements

* docker
* docker-compose

### note

めんどくさい人は `requirements.txt` から install しても OK 

## Quick Start

```bash
docker-compose up -d --build
```

Access to http://localhost:8501/ !!
